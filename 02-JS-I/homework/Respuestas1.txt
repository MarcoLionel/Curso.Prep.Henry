Variable es un código en el que se debe escribir "var", "espacio" y "nombre de la variable" (var+espacio+nombre variable), se le puede asignar un valor utilizando el signo "=", tambien se pueden utilizar variables constantes "const", la cual no se puede modificar. Además cuando se pueden crear muchas variables dentro de los scope que sólo se toman las que estén dentro de éste.
Instrumentos de cuerda son aquellos códigos que se colocan entre comillas
Funciones: Son códigos diseñados para que se realicen determinadas tareas en particular. Se utiliza la palabra "function", la cuál es un código reservado. Seguido del nombre de la función, paréntesis y llaves. Las funciones pueden tener argumentos, que son como variables que declaramos dentro de la función y van a tomar un valor cuando se invoque la misma, estos argumentos se separan com comas.
Declaraciones: Se utilizan para realizar comparaciones con retorno "true" o "false", cabe destacar que para momentos donde el retorno arroja un resultado que es igual, se coloca doblemente el signo "=" (==) para obtener un booleano. Tambien se pueden utilizar expresiones "else" como opción alternativa o complementaria para el retorno. 
Valores Booleanos: Son dos, el retorno siempre va a ser "true" o "false", sería en sistema binario 0 y 1.
